# Requires
1. youtube-dl
2. animated-wallpaper
<https://github.com/Ninlives/animated-wallpaper>

# Installation
1. Move this folder to /usr/local/share/
2. Move the ".desktop" files to /usr/share/applications/
3. Reload GNOME - X11 press f2 and then r
